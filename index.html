Javascript Interview Sample 002

  • select a secret number between 1 and 1,000
  • allow the user to try and guess the number up to 12 times
  • if the user guesses the number, let her know she won
  • if the user runs out of guesses, let her know she lost
  • if the user guesses to high, suggest she guess a lower number
  • if the user guesses to low, suggest she guess a higher number
  • show the user how many guesses are remaining
